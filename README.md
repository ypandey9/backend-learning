# Ecommerce Backend API

Production-ready Node.js backend with:

- JWT Authentication
- Role Based Access Control
- MongoDB Transactions
- Redis Caching
- Background Jobs (BullMQ)
- API Rate Limiting
- Swagger Documentation

## Tech Stack

Node.js
Express.js
MongoDB
Redis
BullMQ
Swagger

## API Documentation

http://localhost:5000/api-docs

## Run Project

npm install
npm run dev
